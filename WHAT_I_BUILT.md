# What I Built — SuccessOS AI

---

## Working Prototype

React + Vite application with 7 fully functional modules and realistic mock data for 8 enterprise SaaS sample accounts.

**To run locally:**
```bash
cd app
npm install
npm run dev
```

---

## Prototype Modules

| Module | What It Demonstrates |
|---|---|
| **Dashboard** | KPI command center — ARR at risk, churn risk count, expansion opportunities, AI insight panel |
| **Accounts** | 8 enterprise accounts with health scores, timelines, and full detail views |
| **Meeting Intelligence** | Paste meeting notes → structured churn/expansion analysis + follow-up email draft |
| **Feature Impact** | Enter a product feature → customer match table with priority scores and AI-generated messaging |
| **Agent Actions** | Approval queue — approve or dismiss AI-recommended actions, live state updates |
| **QBR Generator** | Per-account executive QBR brief with 9 AI-generated sections |
| **Settings** | Agent mode, confidence threshold, approval toggles, mock integration panel |
